# Tech Stack: Mum_Finance

## Core Technologies
- **Language:** [TypeScript](https://www.typescriptlang.org/) - Provides strong typing for more reliable code.
- **Framework:** [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/) - Allows for cross-platform mobile development with a smooth developer experience.
- **Routing:** [Expo Router](https://docs.expo.dev/router/introduction/) - File-based routing for React Native.

## Frontend & UI
- **State Management:** [Zustand](https://github.com/pmndrs/zustand) - A small, fast, and scalable bearbones state-management solution.
- **Styling:** [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native) - Provides utility-first styling for consistent UI.

## Data & Logic
- **CSV Parsing:** [PapaParse](https://www.papaparse.com/) - Robust CSV parser for handling bank statement imports.
- **LLM Integration:** [Cerebras Cloud SDK](https://cerebras.ai/) - Used for high-performance inference to power financial insights.

## Development & Testing
- **Testing:** [Jest](https://jestjs.io/) and [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/) - Comprehensive suite with exhaustive unit, integration, and E2E coverage.
- **Robustness:** >95% line coverage for core logic and state engines, utilizing high-fidelity SDK mocking and resilience testing.
